## lists
- lists are used for all sorts of things :
  - regular bullets and numbered lists
  - navigation
  - organizing other content
  
## two types of list :
1. ordered lists : _ol_
2. unordered lists : _ul_
- each bullet or numbered item is a **list item** _li_
```
<ol>
  <li>list item</li>
  <li>list item</li>
  <li>list item</li>
</ol>
```
